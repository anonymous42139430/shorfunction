# C#ユーザーのためのショアのアルゴリズムの実装
1.このプログラムについて
　このプログラムは、ショアのアルゴリズムを使って入力された数字の因数を見つけるプログラムです。

2.プログラム開発者の意図
　このプログラムは、ショアのアルゴリズムを忠実に再現することを目標として作りました。使用言語はC#です。

3.使い方
　①C#コンパイラでプログラムをコンパイルしアプリケーションを作成します。
　②起動して、因数を見つけたい数字を入力します。
　③何回実行するかを指定します。(回数の数字のみ、「～回」などとは入力しないでください)
　④Enterキーを押します。

4.注意点
・このプログラムは、回数を1(回)としたとすれば1回ランダムな数字を選び、演算の結果答えにならないと分かれば終わってしまうプ
ログラムなので、
　・数字が4~5桁の場合：100~500
　・数字が6~7桁の場合：1000~5000
を目安に回数を設定してください。
・このプログラムは、構造上小さい因数があるほど高確率で見つかるため、例えば偶数であったり3,5,9,11の倍数など比較的見つけ
やすい小さな因数がないか確認の上使われることを推奨します。本プログラムはショアのアルゴリズムを忠実に再現するといったとこ
ろに重点を置いたため、倍数の判定や、同じ因数を複数回出力しないようにするなどといった操作は含んでおりません。
